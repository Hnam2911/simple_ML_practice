# 🤖 Simple ML Practice

Welcome to the **Simple ML Practice** repository! 👋

## 🎯 Repository Purpose
This repository serves as my personal workspace and portfolio for learning Machine Learning and Data Science. Even though these are relatively small practice projects, this repository acts as a notebook that helps me:
* Track my daily learning progress and coding skills.
* Master the usage of fundamental Data Science libraries (like Pandas, NumPy, Scikit-Learn).
* Practice the complete Data Pipeline, from exploratory data analysis (EDA) and preprocessing to model training and prediction.

## 📂 Repository Structure
The repository is organized into two main directories:

### 1. `data/` (Datasets)
This folder contains the raw and processed datasets used for the practice notebooks. 
> ⚠️ **Important Note:** Approximately 50% of the data in this folder was collected from **[Kaggle](https://www.kaggle.com/)**. These datasets are used strictly for personal learning and educational purposes.

Included data files:
* `co2.csv`
* `English_grammar.csv`
* `gold_price.csv`
* `Heart_Disease.csv`
* `StudentPerformance.csv`
* `processed_student_performance.csv`

### 2. `jupyter/` (Practice Notebooks)
This folder contains Jupyter Notebooks (`.ipynb`) dedicated to specific datasets and machine learning problems:
* `co2_forecasting.ipynb`: Time-series forecasting practice focusing on CO2 emissions.
* `english_grammar_classified.ipynb`: Text classification and NLP practice for English grammar.
* `heart_disease.ipynb`: Classification problem to predict the likelihood of heart disease.
* `student_performance.ipynb`: Analysis and prediction of student performance.
* `learn.ipynb`: A draft/scratchpad notebook for free-form coding and testing.

## 🛠️ Technologies & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook / JupyterLab
* **Core Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn (varies by notebook).

## 🚀 How to Use
If you want to clone the code to review or run it on your local machine:

1. **Clone this repository:**
   ```bash
   git clone [https://github.com/Hnam2911/simple_ML_practice.git](https://github.com/Hnam2911/simple_ML_practice.git)
   cd simple_ML_practice
2. **Install necessary libraries:**
Make sure you have Jupyter Notebook and the standard Python libraries for Data Science installed. You can easily install them via pip or conda.

3. **Launch Jupyter Notebook:**
Navigate to the jupyter/ directory and start Jupyter:
   ```bash
   cd jupyter
   jupyter notebook

4. **Data Paths (Important):**
When running the .ipynb files, ensure that the file paths for reading the .csv datasets correctly point to the ../data/ directory.
